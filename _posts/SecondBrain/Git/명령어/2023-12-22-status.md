#Git 
## 🌈 status
+ **파일의 상태**를 확인할 수 있다.
+ Git의 3가지 영역|Working Directory와 Git의 3가지 영역|Staging Area의 상태를 확인하기 위한 명령어이다.
+ 수정된 파일, Area/Git/명령어/commit 가능한 변경 사항, Staging Area에 추가된/추가되지 않은 파일 등을 알 수 있다.
```dos
git status
```
