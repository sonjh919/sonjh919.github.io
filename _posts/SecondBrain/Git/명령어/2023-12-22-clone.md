#Git 
## 🌈 clone
+ 원격 저장소에 있는 프로젝트를 가져온다.
+ 해당 프로젝트를 가져오면 이미 init이 되어 있으며 remote 등록도 자동으로 된다.
+ 기존 작업 내용들은 날아간다. 만약 병합을 원한다면 pull 명령어를 사용하자.

```cs
git clone 원격저장소주소
```
