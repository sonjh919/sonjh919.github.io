#Git 
## 🌈 reset
+ 현재 작업 중인 HEAD의 포인터를 특정 위치로 변경할 수 있다.
+ Area/Git/명령어/commit ID를 이용해 원하는 commit 시점으로 바꿀 수 있다.
```dos
git reset --hard 'commitID'
```
